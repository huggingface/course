# မိတ်ဆက်

Hugging Face သင်တန်းကနေ ကြိုဆိုပါတယ်။ ဒီမိတ်ဆက်က working environment တစ်ခုကို ဘယ်လိုတည်ဆောက်ရမလဲဆိုတာ လမ်းညွှန်ပေးမှာပါ။ သင်တန်းကို အခုမှစတက်မယ်ဆိုရင် [အခန်း ၁] (/course/chapter1) ကို ပထမဆုံးကြည့်ပြီးမှ ပြန်လာပြီး ကိုယ်တိုင်ကုဒ်ရေးကြည့်ဖို့ environment ကို တည်ဆောက်ဖို့ အကြံပြုလိုပါတယ်။

ဒီသင်တန်းမှာ ငါတို့သုံးမယ့် library အားလုံးက Python package တွေအနေနဲ့ ရနိုင်တာမို့ Python environment ကို ဘယ်လိုတည်ဆောက်ရမလဲ နဲ့ လိုအပ်တဲ့ library တွေကို ဘယ်လို install လုပ်ရမလဲဆိုတာ ပြသပေးမှာပါ။

Working environment ကို တည်ဆောက်ဖို့ နည်းလမ်းနှစ်ခုကို ပြောပြမှာပါ။ Colab notebook ဒါမှမဟုတ် Python virtual environment ကိုသုံးပြီးတော့ပေါ့။ ကိုယ်နဲ့ အကိုက်ညီဆုံးတစ်ခုကို ရွေးချယ်လို့ရပါတယ်။ အစပြုသူတွေအတွက် Colab notebook ကိုသုံးပြီး စတင်ဖို့ အထူးအကြံပြုလိုပါတယ်။

Windows system ကိုတော့ ဒီမှာ မပြောပြတော့ဘူးနော်။ Windows သုံးနေတယ်ဆိုရင် Colab notebook ကိုသုံးပြီး လိုက်လုပ်ကြည့်ဖို့ အကြံပြုလိုပါတယ်။ Linux distribution ဒါမှမဟုတ် macOS သုံးနေတယ်ဆိုရင် ဒီမှာဖော်ပြထားတဲ့ နည်းလမ်းတစ်ခုခုကို သုံးလို့ရပါတယ်။

သင်တန်းရဲ့ အများစုက ကိုယ့်မှာ Hugging Face အကောင့်ရှိဖို့ လိုအပ်တယ်။ မရှိသေးရင်၊ အခုတစ်ခုဖန်တီးထားဖို့ အကြံပြုလိုပါတယ်။ [အကောင့်ဖန်တီးပါ] (https://huggingface.co/join)။

## Google Colab notebook တစ်ခုအသုံးပြုခြင်း

Colab notebook ကိုသုံးတာမှာ အရှင်းဆုံးက setup ပါပဲ။ browser မှာ notebook တစ်ခုဖွင့်ပြီး အခုပဲကုဒ်ရေးလိုက်ပါ။

Colab နဲ့ မရင်းနှီးဘူးဆိုရင် [မိတ်ဆက်ခြင်း] (https://colab.research.google.com/notebooks/intro.ipynb) ကို လေ့လာကြည့်ဖို့ အကြံပြုတယ်။ Colab က GPU ဒါမှမဟုတ် TPU လိုမျိုး hardware acceleration တွေကို သုံးခွင့်ပေးတယ်၊ ပြီးတော့ workload သေးသေးလေးတွေအတွက် အခမဲ့ပဲ။

Colab မှာ အသုံးပြုရတာ ကျွမ်းကျင်လာပြီဆိုရင်တော့ စာအုပ်သစ်တစ်အုပ် ဖန်တီးပြီး စတင် setup လိုက်ပါ။

နောက်တစ်ဆင့်က ဒီသင်တန်းမှာ ကျွန်တော်တို့သုံးမယ့် libraries တွေကို install လုပ်ရမှာပါ။ install လုပ်ဖို့အတွက် Python ရဲ့ package manager ဖြစ်တဲ့ `pip` ကိုသုံးပါမယ်။ notebooks တွေမှာ system commands တွေကို `!` နဲ့အစချီပြီး run လို့ရပါတယ်။ ဒါကြောင့် 🤗 Transformers library ကို အောက်ပါအတိုင်း install လုပ်နိုင်ပါတယ်။

```
!pip install transformers
```

သင်ထည့်သွင်းထားတဲ့ package ကို Python runtime မှာ import လုပ်ကြည့်ခြင်းအားဖြင့် မှန်ကန်စွာထည့်သွင်းပြီးမပြီး စစ်ဆေးနိုင်ပါတယ်။

```
import transformers
```

ဒါက 🤗 Transformers ရဲ့ တော်တော်လေး ပေါ့ပါးတဲ့ ဗားရှင်းကို install လုပ်ပေးပါတယ်။ အထူးသဖြင့် ဘယ် machine learning framework တစ်ခုမှ (PyTorch ဒါမှမဟုတ် TensorFlow လို) install လုပ်မှာ မဟုတ်ပါဘူး။ ဒီ library ရဲ့  feature အများကြီးကို ကျွန်တော်တို့ သုံးသွားမှာဖြစ်တဲ့အတွက် development version ကို install လုပ်ဖို့ အကြံပြုပါတယ်။ ဒီ version မှာ စဉ်းစားကြည့်နိုင်သမျှ use case တိုင်းအတွက် လိုအပ်တဲ့ dependency တွေ အားလုံး ပါဝင်ပါတယ်။

```

```

ဒါက အချိန်နည်းနည်းလေး ယူပါလိမ့်မယ်၊ ပြီးရင်တော့ သင်တန်းကျန်တာတွေအတွက် အဆင်ပြေသွားမှာပါ။

## Python virtual environment တစ်ခု အသုံးပြုခြင်း

Python virtual environment ကိုသုံးချင်တယ်ဆိုရင် ပထမဆုံး ကွန်ပျူတာမှာ Python ကို install လုပ်ရပါမယ်။ [ဒီလင့်ခ်](https://realpython.com/installing-python/) ကနေ ဘယ်လို install လုပ်ရမလဲဆိုတာ လေ့လာနိုင်ပါတယ်။

Python install လုပ်ပြီးရင် terminal မှာ Python command တွေကို run နိုင်ပါပြီ။ `python --version` ဆိုတဲ့ command ကို run ပြီး Python ကိုမှန်မှန်ကန်ကန် install လုပ်ပြီးပြီလားဆိုတာ စစ်ဆေးနိုင်ပါတယ်။ ဒီ command က ကွန်ပျူတာမှာရှိတဲ့ Python version ကိုပြပေးပါလိမ့်မယ်။

Terminal မှာ `python --version` လိုမျိုး Python command တွေ run တဲ့အခါ ဒီ command ကို run ပေးနေတဲ့ program က ကွန်ပျူတာရဲ့ "main" Python လို့သဘောထားရပါမယ်။ ဒီ main Python installation မှာ ဘာ package မှ install မလုပ်ဘဲ သီးခြား application တစ်ခုချင်းစီအတွက် သီးခြား environment တွေ သုံးသင့်ပါတယ်။ ဒီလိုလုပ်ခြင်းအားဖြင့် application တစ်ခုချင်းစီမှာ ကိုယ်ပိုင် dependencies တွေ၊ package တွေရှိနိုင်ပြီး တခြား application တွေနဲ့ compatibility ကိစ္စတွေကို စိတ်ပူစရာမလိုတော့ပါဘူး။

Python မှာ ဒါကို [*virtual environments*](https://docs.python.org/3/tutorial/venv.html) တွေသုံးပြီးလုပ်ပါတယ်။ virtual environment တစ်ခုဆိုတာ သီးခြား directory တစ်ခုဖြစ်ပြီး အဲဒီထဲမှာ Python installation တစ်ခုနဲ့ သက်ဆိုင်ရာ application အတွက်လိုအပ်တဲ့ package တွေအားလုံးပါဝင်ပါတယ်။ ဒီလို virtual environment တွေကို ဖန်တီးဖို့အတွက် tool တော်တော်များများရှိပေမယ့် [`venv`](https://docs.python.org/3/library/venv.html#module-venv) ဆိုတဲ့ official Python package ကိုသုံးပါမယ်။

ပထမဆုံး application ထည့်သွင်းမယ့် directory ကိုဖန်တီးပါ။ ဥပမာ - home directory ရဲ့ root မှာ *transformers-course* ဆိုတဲ့ directory အသစ်တစ်ခု ဖန်တီးနိုင်ပါတယ်။

```
mkdir ~/transformers-course
cd ~/transformers-course
```

ဒီဖိုင်တွဲထဲကနေ Python ရဲ့ `venv` module ကိုသုံးပြီး virtual environment တစ်ခု တည်ဆောက်လိုက်ပါ။

```
python -m venv .env
```

အခုဆိုရင် ဖိုင်တွဲထဲမှာ *.env* ဆိုတဲ့ ဖိုင်တွဲအသစ်တစ်ခု ပေါ်လာပါလိမ့်မယ်။

```
ls -a
```

```out
.      ..    .env
```

`activate` နဲ့ `deactivate` scripts တွေကိုသုံးပြီး virtual environment ထဲကို ဝင်လိုက်ထွက်လိုက် လုပ်လို့ရပါတယ်။

```
# Activate the virtual environment
source .env/bin/activate

# Deactivate the virtual environment
deactivate
```

`which python` command ကို run ပြီး environment က activate ဖြစ်နေရဲ့လား စစ်ဆေးလို့ရပါတယ်။ virtual environment ကို ညွှန်ပြနေရင်တော့ activate ဖြစ်သွားပါပြီ။

```
which python
```

```out
/home/<user>/transformers-course/.env/bin/python
```

### Dependencies များ Install လုပ်ခြင်း

Google Colab ကို အသုံးပြုခြင်း အပိုင်းမှာ ဖော်ပြခဲ့သလိုပဲ၊ ဆက်လုပ်ဖို့အတွက် လိုအပ်တဲ့ package များကို ထည့်သွင်းရပါမယ်။ 🤗 Transformers ရဲ့ developer version ကို `pip` package manager ကို သုံးပြီး ထည့်သွင်းနိုင်ပါတယ်။

```
pip install "transformers[sentencepiece]"
```
