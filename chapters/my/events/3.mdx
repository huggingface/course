# Gradio Blocks Party[[gradio-blocks-party]]

သင်တန်း၏ Gradio အခန်း ထွက်ရှိမှုနှင့်အတူ Hugging Face သည် community event တစ်ခုကို လက်ခံကျင်းပခဲ့ပြီး၊ ၎င်းသည် Gradio Blocks feature အသစ်ကို အသုံးပြု၍ machine learning demos ကောင်းများ တည်ဆောက်ခြင်းအကြောင်း ဖြစ်သည်။

community မှ ဖန်တီးထားသော demos များအားလုံးကို Hub ပေါ်ရှိ [`Gradio-Blocks`](https://huggingface.co/Gradio-Blocks) အဖွဲ့အစည်းအောက်တွင် ရှာတွေ့နိုင်ပါသည်။ အနိုင်ရသူများ၏ ဥပမာအချို့မှာ အောက်ပါအတိုင်း ဖြစ်သည်-
