## 🤗 သင်တန်းကနေ ကြိုဆိုပါတယ်။

<Youtube id="00GKzGyWFEs" />

ဤသင်တန်းသည် [Hugging Face](https://huggingface.co/) ecosystem မှ Library များဖြစ်သော [🤗 Transformers](https://github.com/huggingface/transformers), [🤗 Datasets](https://github.com/huggingface/datasets), [🤗 Tokenizers](https://github.com/huggingface/tokenizers),   
 နှင့် [🤗 Accelerate](https://github.com/huggingface/accelerate) တို့အပြင် [Hugging Face Hub](https://huggingface.co/models) ကို အသုံးပြု၍ Natural Language Processing (NLP) အကြောင်း သင်ကြားပေးပါမည်။ ၎င်းသည် အခမဲ့ဖြစ်ပြီး ကြော်ငြာများ မပါဝင်ပါ။


## ဘာတွေ မျှော်လင့်နိုင်မလဲ။

သင်တန်း၏ အကျဉ်းချုပ်မှာ အောက်ပါအတိုင်းဖြစ်သည်။

- အခန်း ၁ မှ ၄ တွင် 🤗 Transformers library ၏ အဓိက အယူအဆများကို မိတ်ဆက်ပေးပါသည်။ သင်တန်း၏ ဤအပိုင်းကို ပြီးဆုံးသောအခါတွင် Transformer မော်ဒယ်များ မည်သို့အလုပ်လုပ်ပုံကို သင် နားလည်ပြီး [Hugging Face Hub](https://huggingface.co/models) မှ မော်ဒယ်တစ်ခုကို မည်သို့အသုံးပြုရမည်၊ ဒေတာစုတစ်ခုတွင် fine-tune လုပ်နည်းနှင့် Hub တွင် သင်၏ရလဒ်များကို မည်သို့မျှဝေရမည်ကို သိရှိပါလိမ့်မည်။
- အခန်း ၅ မှ ၈ တွင် 🤗 Datasets နှင့် 🤗 Tokenizers ၏ အခြေခံများကို သင်ကြားပေးပြီးနောက် NLP လုပ်ငန်းများကို လေ့လာပါမည်။ ဤအပိုင်းကို ပြီးဆုံးသောအခါတွင် အဖြစ်များတဲ့ NLP ပြဿနာများကို သင်ကိုယ်တိုင် ဖြေရှင်းနိုင်ပါလိမ့်မည်။
- အခန်း ၉ မှ ၁၂ တွင် NLP ထက်ကျော်လွန်၍ Transformer မော်ဒယ်များကို Speech processing နှင့် Computer vision တို့တွင် မည်သို့အသုံးပြုနိုင်ပုံကို လေ့လာပါမည်။ သင့်မော်ဒယ်များ၏ demo များကို မည်သို့တည်ဆောက်ရမည်နှင့် မျှဝေရမည်၊ ထုတ်လုပ်မှုပတ်ဝန်းကျင်(production environments) အတွက် မည်သို့ပိုမိုကောင်းမွန်အောင်လုပ်ရမည်ကို သင် လေ့လာရပါလိမ့်မည်။ ဤအပိုင်းကို (အားလုံးနီးပါး) ပြီးဆုံးသောအခါတွင်  မည်သည့် machine learning ကိစ္စတွင်မဆို 🤗 Transformers ကို အသုံးပြုရန် အသင့်ဖြစ်နေပါပြီ။

ဤသင်တန်းသည်

* Python ကို ကောင်းစွာ နားလည်ရန် လိုအပ်သည်။
* [fast.ai's](https://www.fast.ai/) [Practical Deep Learning for Coders](https://course.fast.ai/) သို့မဟုတ် [DeepLearning.AI](https://www.deeplearning.ai/) မှ တီထွင်ထားသော ပရိုဂရမ်များကဲ့သို့သော Deep learning သင်တန်းကို စတင်သင်ယူပြီးနောက် တက်ရောက်ခြင်းသည် ပိုကောင်းပါသည်။
* [PyTorch](https://pytorch.org/) သို့မဟုတ် [TensorFlow](https://www.tensorflow.org/) အကြောင်း ကြိုတင် သိရှိရန် မလိုအပ်သော်လည်း ၎င်းတို့ထဲမှ တစ်ခုခုကို အနည်းငယ် နားလည်ထားခြင်းသည် အထောက်အကူဖြစ်ပါလိမ့်မည်။

ဤသင်တန်းကို ပြီးဆုံးပြီးနောက် DeepLearning.AI ၏ [Natural Language Processing အထူးပြု](https://www.coursera.org/specializations/natural-language-processing?utm_source=deeplearning-ai&utm_medium=institutions&utm_campaign=20211011-nlp-2-hugging_face-page-nlp-refresh)   
 ကို ကြည့်ရှုရန် အကြံပြုလိုပါသည်။ ၎င်းတွင် Naive Bayes နှင့် LSTMs ကဲ့သို့သော Traditional  NLP မော်ဒယ်များစွာ ပါဝင်ပြီး သိရှိထားသင့်ပါသည်။

## မေးလေ့ရှိသောမေးခွန်းများ

မေးလေ့ရှိသောမေးခွန်းများအတွက် အဖြေအချို့ကို ဤနေရာတွင် ဖော်ပြထားပါသည်။

- **ဤသင်တန်းကို တက်ရောက်ခြင်းဖြင့် အသိအမှတ်ပြုလက်မှတ် ရရှိပါသလား။**
လောလောဆယ်တွင် ဤသင်တန်းအတွက် အသိအမှတ်ပြုလက်မှတ် မရှိသေးပါ။ သို့သော် Hugging Face ecosystem အတွက် အသိအမှတ်ပြုလက်မှတ်ပရိုဂရမ်တစ်ခုကို ကျွန်ုပ်တို့ လုပ်ဆောင်နေပါသည် - စောင့်မျှော်ကြည့်ရှုပါ။

- **ဤသင်တန်းတွင် အချိန်မည်မျှ သုံးစွဲသင့်သနည်း။**
ဤသင်တန်းရှိ အခန်းတစ်ခန်းစီကို တစ်ပတ်အတွင်း ပြီးမြောက်အောင် ဒီဇိုင်းထုတ်ထားပြီး တစ်ပတ်လျှင် ၆-၈ နာရီခန့် လုပ်ဆောင်ရန် လိုအပ်ပါသည်။ သို့သော် သင်တန်းပြီးဆုံးရန် လိုအပ်သလောက် အချိန်ယူနိုင်ပါသည်။

- **မေးခွန်းတစ်ခုခုရှိရင် ဘယ်မှာမေးလို့ရမလဲ။**
သင်တန်း၏ အပိုင်းတစ်ခုခုနှင့်ပတ်သက်၍ မေးခွန်းတစ်ခုခုရှိပါက စာမျက်နှာ၏ ထိပ်ရှိ "*Ask a question*" banner ကို နှိပ်ပါက [Hugging Face forums](https://discuss.huggingface.co/) ၏ သက်ဆိုင်ရာ အပိုင်းသို့ အလိုအလျောက် ပြန်ညွှန်းပေးပါမည်။

## စတင်ကြစို့
အသင့်ဖြစ်ပြီလား။ ဤအခန်းတွင် သင်သည် အောက်ပါတို့ကို လေ့လာရပါမည်။

* Text generation နှင့် classification ကဲ့သို့သော NLP လုပ်ငန်းများကို ဖြေရှင်းရန် `pipeline()` function ကို အသုံးပြုနည်း
* Transformer architecture(ပုံစံ ဖွဲ့စည်းပုံ) အကြောင်း
* Encoder၊ decoder နှင့် encoder-decoder architecture များနှင့် အသုံးပြုမှုကိစ္စရပ်များကို မည်သို့ခွဲခြားရမည်နည်း
