یہ واقعی 🤗 Datasets لائبریری کے ذریعے ایک دلچسپ سفر تھا — اس مرحلے تک پہنچنے پر آپ کو مبارکباد! اس باب سے حاصل کردہ علم کی مدد سے آپ کو یہ کرنے کے قابل ہونا چاہیے:

- Hugging Face Hub، آپ کے لیپ ٹاپ، یا آپ کی کمپنی کے کسی remote server سے کہیں بھی ڈیٹاسیٹس لوڈ کرنا۔
- `Dataset.map()` اور `Dataset.filter()` فنکشنز کے ملاپ کا استعمال کرتے ہوئے اپنے ڈیٹا کو منظم کرنا۔
- `Dataset.set_format()` کا استعمال کرتے ہوئے Pandas اور NumPy جیسے ڈیٹا فارمیٹس کے درمیان تیزی سے تبدیلی کرنا۔
- اپنا خود کا ڈیٹاسیٹ تخلیق کرنا اور اسے Hugging Face Hub پر push کرنا۔
- Transformer ماڈل کا استعمال کرتے ہوئے اپنے documents کو embed کرنا اور FAISS کا استعمال کرتے ہوئے ایک semantic search engine تیار کرنا۔

[باب 7](/course/chapter7) میں، ہم ان تمام بنیادی NLP tasks میں گہرائی سے جائیں گے جن کے لیے Transformer ماڈلز بہترین ہیں۔ آگے بڑھنے سے پہلے، 🤗 Datasets کے بارے میں اپنے علم کا امتحان لینے کے لیے ایک quick quiz ضرور آزمائیں!
