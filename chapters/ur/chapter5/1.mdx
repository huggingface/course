[باب 3](/course/chapter3) میں، آپ نے پہلی بار 🤗 Datasets لائبریری کے بارے میں سیکھا اور دیکھا کہ ماڈل کو فائن ٹیون کرنے کے تین اہم مراحل ہوتے ہیں:

1. Hugging Face Hub سے ایک ڈیٹاسیٹ لوڈ کریں۔
2. `Dataset.map()` کا استعمال کرکے ڈیٹا کو پروسیس کریں۔
3. میٹرکس کو لوڈ اور کمپیوٹ کریں۔

لیکن یہ تو بس شروعات ہے! 🤗 Datasets لائبریری میں اور بھی بہت کچھ ہے۔ اس باب میں، ہم اس لائبریری کو تفصیل سے سمجھیں گے اور درج ذیل سوالات کے جوابات تلاش کریں گے:

* اگر آپ کا ڈیٹاسیٹ Hub پر موجود نہ ہو تو کیا کریں؟
* ڈیٹاسیٹ کو کس طرح مختلف حصوں میں تقسیم کریں؟ (اور اگر آپ کو واقعی Pandas کی ضرورت ہو؟)
* اگر ڈیٹاسیٹ بہت بڑا ہو اور آپ کے لیپ ٹاپ کی RAM جواب دے جائے تو کیا کریں؟
* "میموری میپنگ" اور Apache Arrow کیا ہیں؟
* آپ اپنا ڈیٹاسیٹ کیسے بنا کر Hub پر اپلوڈ کر سکتے ہیں؟

یہ تکنیکیں آپ کو جدید ٹوکنائزیشن اور فائن ٹیوننگ کے کاموں کے لیے تیار کریں گی، جو [باب 6](/course/chapter6) اور [باب 7](/course/chapter7) میں شامل ہیں۔ تو ایک کپ کافی لے آئیں اور چلیں شروع کریں!
