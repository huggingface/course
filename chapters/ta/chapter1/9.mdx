நீங்கள் உற்பத்தியில் முன்-பயிற்சிவிக்கப்பட்ட மாடல் அல்லது சிறப்பாக சரிசெய்யப்பட்ட பதிப்பைப் பயன்படுத்த எண்ணம் இருந்தால், கவனத்தில் கொள்ளுங்கள், இந்த மாடல்கள் சக்தி வாய்ந்த கருவிகளாக இருந்தாலும், அவை வரம்புகளுடன் வருகின்றன. இவற்றில் மிகப்பெரியது என்னவென்றால், பெரிய அளவிலான தரவுகளில் முன்-பயிற்சியைக்காக்க அனுமதிப்பதற்காக, ஆராய்ச்சியாளர்கள் அடிக்கடி அவர்களால் காண முடிந்த அனைத்து உள்ளடக்கத்தையும் திரட்டுகிறார்கள், இணையத்தில் கிடைக்கும் நல்லதையும் கெட்டதையும் எடுத்துக்கொள்கிறார்கள்.

ஒரு சிறிய விளக்கத்தைக் கொடுக்க, BERT மாடலுடன் `fill-mask` பைப்லைன் உதாரணத்திற்குத் திரும்பிச் செல்வோம்:

```python
from transformers import pipeline

unmasker = pipeline("fill-mask", model="bert-base-uncased")
result = unmasker("This man works as a [MASK].")
print([r["token_str"] for r in result])

result = unmasker("This woman works as a [MASK].")
print([r["token_str"] for r in result])
```

```python out
['carpenter', 'lawyer', 'farmer', 'doctor', 'waiter']
['nurse', 'waitress', 'teacher', 'maid', 'prostitute']
```

ஆணின் தொழிலைக் கணிக்கும்போது, BERT மாடல் எந்தவிதமான சர்ச்சைக்குரிய தேர்வையும் கொடுக்கவில்லை, ஆனால் பெண்ணின் தொழிலைக் கணிக்கும்போது, அது "prostitute" (விலைமாது) என்ற வார்த்தையை சாத்தியமான ஒன்றாகக் காட்டுகிறது. இது போன்ற சார்புநிலைகள் மொழி மாதிரிகளில் பொதுவானவை மற்றும் அவை பயிற்சி பெற்ற தரவுகளிலிருந்து வருகின்றன. இந்த மாதிரிகளைப் பயன்படுத்தும்போது, அவற்றை மனதில் வைத்திருப்பது முக்கியம்.

ஹக்கிங் ஃபேஸ் ஹப்பில் ஒரு மாடலைப் பார்க்கும்போது, அதன் மாடல் கார்டைப் பார்ப்பது ஒரு நல்ல நடைமுறை. இது மாடலின் வரம்புகள் மற்றும் சார்புநிலைகளைப் பற்றி மேலும் அறிய உதவும். ஒரு சமூகப் பொறுப்புள்ள AI உருவாக்குநராக, உங்கள் மாடல்களின் சார்புநிலைகளை வெளிப்படையாக அறிவிப்பது உங்கள் கடமையாகும்.

அடுத்த அத்தியாயத்தில், ஹக்கிங் ஃபேஸ் ஹப்பில் உள்ள மாடல்கள், தரவுத்தொகுப்புகள் மற்றும் டெமோக்களை எவ்வாறு பகிர்வது என்பதைப் பார்ப்போம்.

```python out
['lawyer', 'carpenter', 'doctor', 'waiter', 'mechanic']
['nurse', 'waitress', 'teacher', 'maid', 'prostitute']
```

Intha irandu vaakkiyangalilum vitta vaarthaiyai nirappumpadi ketkappattapothu, model ore oru paalina-podhuvaana pathilai (waiter/waitress) mattume kodukkirathu. Matravai, oru kurippitta paalinathudan thodarbudaiya velaigal aagum -- aam, "pen" matrum "velai" enbathudan model thodarbu படுத்தும் muthal 5 saatthiyakkurugalil prostitute-um idam petrirukkirathu. BERT, inaiyam muzhuvathumirunthu tharavugalai thiratti uruvaakkappadaatha arithaana Transformer model-galil ondraaga irunthapothilum, velippadaiyaga nadunilaiyaana tharavugalai payanpaduthinaalum (ithu [English Wikipedia](https://huggingface.co/datasets/wikipedia) matrum [BookCorpus](https://huggingface.co/datasets/bookcorpus) tharavu thoguppugalil payirvikkappattullathu) ippadi nadakkirathu.

Neengal intha karuvigalai payanpaduthum pothu, neengal payanpaduthum asli model paalina, inaveṟi, allathu orinachcherkkai veruppu ulladakkathai mikavum elithaaga uruvaakka mudiyum enbathai ungal manathin oru moolaiyil vaithirukka vendum. Ungal tharavil model-ai sirappaga seriseyvathu intha ulliruntha saarbநிலைyai pokkaathu.
