# рЃерЃћрЃАрЃљрЃЋрЃљрЃџрЃў

рЃЎрЃћрЃЌрЃўрЃџрЃў рЃўрЃДрЃЮрЃА рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃЏрЃЮрЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљ Hugging Face-рЃўрЃА рЃЎрЃБрЃарЃАрЃќрЃћ! рЃћрЃА рЃЌрЃљрЃЋрЃў рЃЊрЃљрЃњрЃћрЃ«рЃЏрЃљрЃарЃћрЃЉрЃљрЃЌ рЃЋрЃўрЃарЃбрЃБрЃљрЃџрЃБрЃарЃў рЃАрЃљрЃЏрЃБрЃерЃљрЃЮ рЃАрЃўрЃЋрЃарЃфрЃўрЃА рЃњрЃљрЃЏрЃљрЃарЃЌрЃЋрЃљрЃерЃў. рЃЌрЃБ рЃЌрЃЦрЃЋрЃћрЃю рЃћрЃ«рЃџрЃљрЃ«рЃљрЃюрЃА рЃЊрЃљрЃўрЃгрЃДрЃћрЃЌ рЃЎрЃБрЃарЃАрЃў, рЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃўрЃарЃћрЃЉрЃБрЃџрЃўрЃљ рЃърЃўрЃарЃЋрЃћрЃџ рЃарЃўрЃњрЃерЃў рЃњрЃљрЃЊрЃљрЃ«рЃћрЃЊрЃЮрЃЌ [рЃърЃўрЃарЃЋрЃћрЃџ рЃЌрЃљрЃЋрЃА](/course/chapter1), рЃЊрЃљ рЃерЃћрЃЏрЃЊрЃћрЃњ рЃЊрЃљрЃБрЃЉрЃарЃБрЃюрЃЊрЃћрЃЌ рЃерЃћрЃАрЃљрЃЋрЃљрЃџрЃА 
рЃарЃљрЃЌрЃљ рЃЏрЃљрЃарЃбрЃўрЃЋрЃљрЃЊ рЃерЃћрЃФрЃџрЃЮрЃЌ рЃЎрЃЮрЃЊрЃўрЃюрЃњрЃўрЃА рЃЊрЃљрЃгрЃДрЃћрЃЉрЃљ. 

рЃљрЃЏ рЃЎрЃБрЃарЃАрЃерЃў рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃў рЃДрЃЋрЃћрЃџрЃљ рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃљ рЃ«рЃћрЃџрЃЏрЃўрЃАрЃљрЃгрЃЋрЃЊрЃЮрЃЏрЃўрЃљ Python-рЃўрЃА рЃЏрЃЮрЃЊрЃБрЃџрЃћрЃЉрЃўрЃА рЃАрЃљрЃ«рЃўрЃЌ, рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃљрЃЊ рЃљрЃЏ рЃЌрЃљрЃЋрЃерЃў рЃњрЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃЌ рЃЌрЃБ рЃарЃЮрЃњрЃЮрЃа рЃњрЃљрЃЋрЃЏрЃљрЃарЃЌрЃЮрЃЌ Python-рЃўрЃА рЃЋрЃўрЃарЃбрЃБрЃљрЃџрЃБрЃарЃў рЃњрЃљрЃарЃћрЃЏрЃЮ, рЃЊрЃљ рЃЊрЃљрЃЋрЃљрЃўрЃюрЃАрЃбрЃљрЃџрЃўрЃарЃЮрЃЌ рЃАрЃљрЃГрЃўрЃарЃЮ рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃћрЃЉрЃў. 

рЃЕрЃЋрЃћрЃю рЃњрЃўрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃЌ рЃЮрЃа рЃњрЃќрЃљрЃА рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃАрЃљрЃЏрЃБрЃерЃљрЃЮ рЃАрЃўрЃЋрЃарЃфрЃўрЃА рЃњрЃљрЃЏрЃљрЃарЃЌрЃЋрЃўрЃАрЃЌрЃЋрЃўрЃА, Colab Notebook-рЃўрЃА, рЃљрЃю Python-рЃўрЃА рЃЋрЃўрЃарЃбрЃБрЃљрЃџрЃБрЃарЃў рЃњрЃљрЃарЃћрЃЏрЃЮрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃўрЃЌ. рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃЌрЃљрЃЋрЃўрЃАрЃБрЃцрЃџрЃљрЃЊ рЃљрЃўрЃарЃЕрЃўрЃЮрЃЌ рЃўрЃА рЃЋрЃћрЃарЃАрЃўрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЌрЃЦрЃЋрЃћрЃюрЃЌрЃЋрЃўрЃА рЃБрЃцрЃарЃЮ рЃЎрЃЮрЃЏрЃцрЃЮрЃарЃбрЃБрЃџрЃўрЃљ. рЃЌрЃБрЃЏрЃфрЃљ, рЃЕрЃЋрЃћрЃюрЃў рЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃљрЃфрЃўрЃљ рЃЊрЃљрЃЏрЃгрЃДрЃћрЃЉрЃЌрЃљрЃЌрЃЋрЃўрЃА рЃљрЃарЃўрЃА рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃЮрЃЌ Colab Notebook. 

рЃњрЃЌрЃ«рЃЮрЃЋрЃЌ рЃњрЃљрЃўрЃЌрЃЋрЃљрЃџрЃўрЃАрЃгрЃўрЃюрЃЮрЃЌ, рЃарЃЮрЃЏ рЃбрЃБрЃбрЃЮрЃарЃўрЃљрЃџрЃў рЃљрЃа рЃЏрЃЮрЃўрЃфрЃљрЃЋрЃА Windows рЃАрЃўрЃАрЃбрЃћрЃЏрЃљрЃА. рЃЌрЃБ рЃЌрЃЦрЃЋрЃћрЃю рЃўрЃДрЃћрЃюрЃћрЃЉрЃЌ Windows-рЃА, рЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃўрЃарЃћрЃЉрЃБрЃџрЃўрЃљ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃЮрЃЌ Colab Notebook. рЃўрЃЏ рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў, рЃЌрЃБ рЃўрЃДрЃћрЃюрЃћрЃЉрЃЌ Linux-рЃўрЃА рЃарЃЮрЃЏрЃћрЃџрЃўрЃЏрЃћ рЃЊрЃўрЃАрЃбрЃарЃўрЃЉрЃБрЃфрЃўрЃљрЃА, рЃљрЃюрЃљрЃф macOS-рЃА, рЃЌрЃљрЃЋрЃўрЃАрЃБрЃцрЃџрЃљрЃЊ рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃЮрЃарЃўрЃЋрЃћ рЃЏрЃћрЃЌрЃЮрЃЊрЃў рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃЮрЃЌ. 

рЃЎрЃБрЃарЃАрЃўрЃА рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃюрЃљрЃгрЃўрЃџрЃў рЃЏрЃЮрЃўрЃЌрЃ«рЃЮрЃЋрЃА рЃњрЃЦрЃЮрЃюрЃЊрЃћрЃЌ Hugging Face рЃљрЃюрЃњрЃљрЃарЃўрЃерЃў. рЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃўрЃарЃћрЃЉрЃБрЃџрЃўрЃљ рЃЏрЃўрЃАрЃў рЃЊрЃљрЃАрЃљрЃгрЃДрЃўрЃАрЃерЃўрЃЋрЃћ рЃерЃћрЃЦрЃЏрЃюрЃљ: [рЃљрЃюрЃњрЃљрЃарЃўрЃерЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ](https://huggingface.co/join)

## Google Colab notebook-рЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ

Google Colab notebook-рЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ рЃБрЃЏрЃљрЃарЃбрЃўрЃЋрЃћрЃАрЃўрЃљ. рЃБрЃЉрЃарЃљрЃџрЃЮрЃЊ рЃњрЃљрЃ«рЃАрЃћрЃюрЃўрЃЌ рЃўрЃА рЃЌрЃЦрЃЋрЃћрЃюрЃА рЃЋрЃћрЃЉ рЃЉрЃарЃљрЃБрЃќрЃћрЃарЃерЃў, рЃЊрЃљ рЃЊрЃљрЃЋрЃўрЃгрЃДрЃЮрЃЌ рЃЎрЃЮрЃЊрЃўрЃюрЃњрЃў!

рЃЌрЃБ рЃЌрЃЦрЃЋрЃћрЃю рЃљрЃа рЃњрЃљрЃЏрЃЮрЃњрЃўрЃДрЃћрЃюрЃћрЃЉрЃўрЃљрЃЌ Google Colab, рЃњрЃўрЃарЃЕрЃћрЃЋрЃЌ рЃЊрЃљрЃўрЃгрЃДрЃЮрЃЌ рЃЏрЃўрЃАрЃў рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ рЃерЃћрЃЏрЃЊрЃћрЃњрЃў [рЃерЃћрЃАрЃљрЃЋрЃљрЃџрЃў/рЃбрЃБрЃбрЃЮрЃарЃўрЃљрЃџрЃўрЃЌ](https://colab.research.google.com/notebooks/intro.ipynb). Colab рЃњрЃљрЃФрЃџрЃћрЃЋрЃЌ рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃЮрЃЌ рЃљрЃЏрЃљрЃЕрЃЦрЃљрЃарЃћрЃЉрЃћрЃџрЃў рЃљрЃърЃљрЃарЃљрЃбрЃБрЃарЃљ, GPU рЃљрЃю TPU, рЃЊрЃљ рЃЏрЃўрЃАрЃў рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ рЃЏрЃфрЃўрЃарЃћ рЃЊрЃљрЃбрЃЋрЃўрЃарЃЌрЃЋрЃўрЃА рЃарЃћрЃЪрЃўрЃЏрЃерЃў рЃБрЃцрЃљрЃАрЃЮрЃљ.  

Colab-рЃЌрЃљрЃю рЃЊрЃљрЃЏрЃћрЃњрЃЮрЃЉрЃарЃћрЃЉрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ, рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ рЃљрЃ«рЃљрЃџрЃў Notebook рЃЊрЃљ рЃЊрЃљрЃЋрЃўрЃгрЃДрЃЮрЃЌ рЃАрЃљрЃЏрЃБрЃерЃљрЃЮ рЃАрЃўрЃЋрЃарЃфрЃўрЃА рЃњрЃљрЃЏрЃљрЃарЃЌрЃЋрЃљ:

рЃерЃћрЃЏрЃЊрЃћрЃњрЃў рЃюрЃљрЃЉрЃўрЃ»рЃўрЃљ рЃљрЃЏ рЃЎрЃБрЃарЃАрЃўрЃАрЃљрЃЌрЃЋрЃўрЃА рЃАрЃљрЃГрЃўрЃарЃЮ рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃћрЃЉрЃўрЃА рЃЊрЃљрЃўрЃюрЃАрЃбрЃљрЃџрЃўрЃарЃћрЃЉрЃљ. рЃўрЃюрЃАрЃбрЃљрЃџрЃљрЃфрЃўрЃўрЃАрЃЌрЃЋрЃўрЃА рЃњрЃљрЃЏрЃЮрЃЋрЃўрЃДрЃћрЃюрЃћрЃЉрЃЌ `pip`-рЃА, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф Python-рЃўрЃА рЃърЃљрЃЎрЃћрЃбрЃћрЃЉрЃўрЃА рЃЏрЃћрЃюрЃћрЃ»рЃћрЃарЃўрЃљ. Notebook-рЃћрЃЉрЃерЃў рЃАрЃўрЃАрЃбрЃћрЃЏрЃБрЃарЃў рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃћрЃЉрЃўрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃўрЃАрЃљрЃЌрЃЋрЃўрЃА рЃАрЃљрЃГрЃўрЃарЃЮрЃљ `!` рЃърЃарЃћрЃцрЃўрЃЦрЃАрЃўрЃА рЃЊрЃљрЃарЃЌрЃЋрЃљ, рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃљрЃЊ ­ЪцЌ Transformers рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃўрЃА рЃЊрЃљрЃўрЃюрЃАрЃбрЃљрЃџрЃўрЃарЃћрЃЉрЃўрЃАрЃљрЃЌрЃЋрЃўрЃА рЃњрЃљрЃЏрЃЮрЃЋрЃўрЃДрЃћрЃюрЃћрЃЉрЃЌ рЃерЃћрЃЏрЃЊрЃћрЃњ рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљрЃА:

```
!pip install transformers
```


рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃгрЃљрЃарЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌ рЃЊрЃљрЃўрЃюрЃАрЃбрЃљрЃџрЃўрЃарЃћрЃЉрЃўрЃА рЃњрЃљрЃЊрЃљрЃАрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃџрЃљрЃЊ рЃърЃљрЃЎрЃћрЃбрЃў рЃЊрЃљрЃљрЃўрЃЏрЃърЃЮрЃарЃбрЃЮрЃЌ Python-рЃерЃў: 
```
import transformers
```

рЃќрЃћрЃЏрЃЮрЃЌрЃљ рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљ рЃљрЃўрЃюрЃАрЃбрЃљрЃџрЃўрЃарЃћрЃЉрЃА ­ЪцЌ Transformers рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃўрЃА рЃЏрЃАрЃБрЃЉрЃБрЃЦ рЃЋрЃћрЃарЃАрЃўрЃљрЃА. рЃЎрЃћрЃарЃФрЃЮрЃЊ, рЃљрЃарЃфрЃћрЃарЃЌрЃў рЃЏрЃљрЃюрЃЦрЃљрЃюрЃБрЃарЃў рЃАрЃгрЃљрЃЋрЃџрЃћрЃЉрЃўрЃА рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃљ (рЃЏрЃљрЃњ.: Pytorch рЃљрЃю Tensorflow) рЃљрЃа рЃўрЃюрЃАрЃбрЃљрЃџрЃўрЃарЃЊрЃћрЃЉрЃљ. рЃарЃљрЃЊрЃњрЃљрЃю рЃЕрЃЋрЃћрЃю рЃЊрЃљрЃњрЃЋрЃГрЃўрЃарЃЊрЃћрЃЉрЃљ рЃњрЃљрЃЏрЃЮрЃЋрЃўрЃДрЃћрЃюрЃЮрЃЌ рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃЎрЃўрЃА рЃЉрЃћрЃЋрЃарЃў рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃљрЃџрЃў, рЃњрЃўрЃарЃЕрЃћрЃЋрЃЌ рЃЊрЃљрЃљрЃўрЃюрЃАрЃбрЃљрЃџрЃўрЃарЃЮрЃЌ рЃЊрЃћрЃЋрЃћрЃџрЃЮрЃърЃЏрЃћрЃюрЃб рЃЋрЃћрЃарЃАрЃўрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃЊрЃљрЃљрЃўрЃюрЃАрЃбрЃљрЃџрЃўрЃарЃћрЃЉрЃА рЃДрЃЋрЃћрЃџрЃљ рЃЊрЃљрЃЏрЃЮрЃЎрЃўрЃЊрЃћрЃЉрЃБрЃџ рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃљрЃА рЃюрЃћрЃЉрЃўрЃАрЃЏрЃўрЃћрЃарЃў рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃА рЃарЃћрЃЪрЃўрЃЏрЃўрЃАрЃљрЃЌрЃЋрЃўрЃА. 
```
!pip install transformers[sentencepiece]
```

рЃћрЃА рЃерЃћрЃЊрЃљрЃарЃћрЃЉрЃўрЃЌ рЃЏрЃћрЃб рЃЊрЃарЃЮрЃА рЃгрЃљрЃўрЃдрЃћрЃЉрЃА, рЃЌрЃБрЃЏрЃфрЃљ рЃЊрЃљрЃАрЃарЃБрЃџрЃћрЃЉрЃўрЃАрЃЌрЃљрЃюрЃљрЃЋрЃћ рЃЌрЃЦрЃЋрЃћрЃю рЃЏрЃќрЃљрЃЊ рЃўрЃЦрЃюрЃћрЃЉрЃўрЃЌ рЃЏрЃЌрЃћрЃџрЃў рЃЎрЃБрЃарЃАрЃўрЃАрЃљрЃЌрЃЋрЃўрЃА! 

## Python-рЃўрЃА рЃЋрЃўрЃарЃбрЃБрЃљрЃџрЃБрЃарЃў рЃњрЃљрЃарЃћрЃЏрЃЮрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ

рЃЌрЃБ рЃЌрЃЦрЃЋрЃћрЃю рЃњрЃўрЃарЃЕрЃћрЃЋрЃюрЃўрЃљрЃЌ Python-рЃўрЃА рЃЋрЃўрЃарЃбрЃБрЃљрЃџрЃБрЃарЃў рЃњрЃљрЃарЃћрЃЏрЃЮрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ, рЃърЃўрЃарЃЋрЃћрЃџ рЃарЃўрЃњрЃерЃў рЃАрЃљрЃГрЃўрЃарЃЮрЃљ рЃЊрЃљрЃљрЃўрЃюрЃАрЃбрЃљрЃџрЃўрЃарЃЮрЃЌ Python рЃЌрЃЦрЃЋрЃћрЃюрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃљрЃерЃў. рЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃўрЃарЃћрЃЉрЃБрЃџрЃўрЃљ рЃљрЃЏ [рЃбрЃБрЃбрЃЮрЃарЃўрЃљрЃџрЃўрЃЌ](https://realpython.com/installing-python/) рЃЊрЃљрЃгрЃДрЃћрЃЉрЃљ.

Python-рЃўрЃА рЃЊрЃљрЃўрЃюрЃАрЃбрЃљрЃџрЃўрЃарЃћрЃЉрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ рЃЏрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљрЃА рЃбрЃћрЃарЃЏрЃўрЃюрЃљрЃџрЃўрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃўрЃЌрЃљрЃф рЃерЃћрЃФрЃџрЃћрЃЉрЃЌ. рЃЊрЃљрЃгрЃДрЃћрЃЉрЃўрЃАрЃљрЃЌрЃЋрЃўрЃА, рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃбрЃћрЃарЃЏрЃўрЃюрЃљрЃџрЃерЃў рЃњрЃљрЃБрЃерЃЋрЃљрЃЌ рЃерЃћрЃЏрЃЊрЃћрЃњрЃў рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЊрЃљрЃњрЃћрЃ«рЃЏрЃљрЃарЃћрЃЉрЃљрЃЌ рЃърЃўрЃЌрЃЮрЃюрЃўрЃА рЃўрЃюрЃАрЃбрЃљрЃџрЃљрЃфрЃўрЃўрЃА рЃгрЃљрЃарЃЏрЃљрЃбрЃћрЃЉрЃўрЃА рЃњрЃљрЃЊрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃљрЃерЃў: `python --version`, рЃарЃљрЃЌрЃљ рЃерЃћрЃФрЃџрЃЮрЃЌ рЃерЃћрЃЏрЃЊрЃћрЃњ рЃћрЃбрЃљрЃърЃћрЃЉрЃќрЃћ рЃњрЃљрЃЊрЃљрЃАрЃЋрЃџрЃљ. рЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃўрЃарЃћрЃЉрЃБрЃџрЃўрЃљ рЃЏрЃЌрЃљрЃЋрЃљрЃа рЃАрЃљрЃЏрЃБрЃерЃљрЃЮ рЃњрЃљрЃарЃћрЃЏрЃЮрЃерЃў рЃљрЃа рЃЊрЃљрЃљрЃўрЃюрЃАрЃбрЃљрЃџрЃўрЃарЃЮрЃЌ рЃърЃљрЃЎрЃћрЃбрЃћрЃЉрЃў, рЃљрЃЏрЃўрЃАрЃљрЃЌрЃЋрЃўрЃА рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃерЃћрЃЦрЃЏрЃюрЃљрЃЌ рЃАрЃърЃћрЃфрЃўрЃцрЃўрЃБрЃарЃў рЃЋрЃўрЃарЃбрЃБрЃљрЃџрЃБрЃарЃў рЃњрЃљрЃарЃћрЃЏрЃЮ рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃў рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃћрЃЉрЃўрЃАрЃљрЃЌрЃЋрЃўрЃА, рЃарЃљрЃЌрЃљ рЃЌрЃљрЃЋрЃўрЃЊрЃљрЃю рЃљрЃўрЃарЃўрЃЊрЃЮрЃЌ рЃърЃљрЃЎрЃћрЃбрЃћрЃЉрЃА рЃерЃЮрЃарЃўрЃА рЃЌрЃљрЃЋрЃАрЃћрЃЉрЃљрЃЊрЃЮрЃЉрЃўрЃА рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃћрЃЉрЃў. 

рЃърЃўрЃЌрЃЮрЃюрЃерЃў рЃљрЃЏрЃўрЃАрЃў рЃњрЃљрЃЎрЃћрЃЌрЃћрЃЉрЃљ рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ [*рЃЋрЃўрЃарЃбрЃБрЃљрЃџрЃБрЃарЃў рЃњрЃљрЃарЃћрЃЏрЃЮрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃўрЃЌ*](https://docs.python.org/3/tutorial/venv.html), рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЊрЃљрЃ«рЃБрЃарЃБрЃџрЃў рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљрЃљ рЃЊрЃљ рЃерЃћрЃўрЃфрЃљрЃЋрЃА рЃАрЃърЃћрЃфрЃўрЃцрЃўрЃБрЃарЃў рЃърЃўрЃЌрЃЮрЃюрЃўрЃА рЃЋрЃћрЃарЃАрЃўрЃљрЃА рЃЏрЃўрЃАрЃў рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃў рЃАрЃљрЃГрЃўрЃарЃЮ рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃћрЃЉрЃўрЃА рЃЋрЃћрЃарЃАрЃўрЃћрЃЉрЃўрЃЌ. рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃў рЃЋрЃўрЃарЃбрЃБрЃљрЃџрЃБрЃарЃў рЃњрЃљрЃарЃћрЃЏрЃЮрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃњрЃќрЃўрЃЌ рЃљрЃарЃўрЃА рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃў, рЃЌрЃБрЃЏрЃфрЃљ рЃЕрЃЋрЃћрЃю рЃњрЃљрЃЏрЃЮрЃЋрЃўрЃДрЃћрЃюрЃћрЃЉрЃЌ рЃърЃўрЃЌрЃЮрЃюрЃўрЃА рЃЮрЃцрЃўрЃфрЃўрЃљрЃџрЃБрЃа рЃърЃљрЃЎрЃћрЃбрЃА: [`venv`](https://docs.python.org/3/library/venv.html#module-venv).

рЃърЃўрЃарЃЋрЃћрЃџ рЃарЃўрЃњрЃерЃў, рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљ рЃАрЃљрЃЊрЃљрЃф рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃўрЃА рЃцрЃљрЃўрЃџрЃћрЃЉрЃў рЃњрЃўрЃюрЃЊрЃљрЃЌ рЃерЃћрЃўрЃюрЃљрЃ«рЃЮрЃЌ - рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ, рЃЌрЃБ рЃњрЃўрЃюрЃЊрЃљрЃЌ рЃарЃЮрЃЏ рЃерЃћрЃЦрЃЏрЃюрЃљрЃЌ рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљ рЃАрЃљрЃ«рЃћрЃџрЃљрЃЊ transformers-course рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃАрЃўрЃАрЃбрЃћрЃЏрЃўрЃА рЃЏрЃЌрЃљрЃЋрЃљрЃа рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљрЃерЃў: 
```
mkdir ~/transformers-course
cd ~/transformers-course
```

рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљрЃерЃў рЃерЃћрЃАрЃЋрЃџрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ рЃЎрЃў рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ рЃЋрЃўрЃарЃбрЃБрЃљрЃџрЃБрЃарЃў рЃњрЃљрЃарЃћрЃЏрЃЮ рЃърЃўрЃЌрЃЮрЃюрЃўрЃА `venv` рЃЏрЃЮрЃЊрЃБрЃџрЃўрЃА рЃЊрЃљрЃ«рЃЏрЃљрЃарЃћрЃЉрЃўрЃЌ:
```
python -m venv .env
```

рЃљрЃЏрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ рЃБрЃюрЃЊрЃљ рЃ«рЃћрЃЊрЃљрЃЋрЃЊрЃћрЃЌ рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљрЃА рЃАрЃљрЃ«рЃћрЃџрЃљрЃЊ *.env* рЃљрЃЦрЃљрЃЏрЃЊрЃћ рЃфрЃљрЃарЃўрЃћрЃџ рЃцрЃЮрЃџрЃЊрЃћрЃарЃерЃў:
```
ls -a
```

```out
.      ..    .env
```

рЃЋрЃўрЃарЃбрЃБрЃљрЃџрЃБрЃарЃў рЃњрЃљрЃарЃћрЃЏрЃЮрЃА рЃљрЃЦрЃбрЃўрЃЋрЃљрЃфрЃўрЃљ рЃљрЃю рЃЊрЃћрЃљрЃЦрЃбрЃўрЃЋрЃљрЃфрЃўрЃљ рЃЏрЃљрЃарЃбрЃўрЃЋрЃљрЃЊ рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ `activate` рЃЊрЃљ `deactivate` рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃћрЃЉрЃўрЃЌ:
```
# рЃЋрЃўрЃарЃбрЃБрЃљрЃџрЃБрЃарЃў рЃњрЃљрЃарЃћрЃЏрЃЮрЃА рЃљрЃЦрЃбрЃўрЃЋрЃљрЃфрЃўрЃљ
source .env/bin/activate

# рЃЋрЃўрЃарЃбрЃБрЃљрЃџрЃБрЃарЃў рЃњрЃљрЃарЃћрЃЏрЃЮрЃА рЃЊрЃћрЃљрЃЦрЃбрЃўрЃЋрЃљрЃфрЃўрЃљ
```
