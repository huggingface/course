# ස්වභාවික භාෂා සැකසීම[[natural-language-processing]]

<CourseFloatingBanner chapter={1} classNames="absolute z-10 right-0 top-0" />

පරිවර්තක ආකෘති සදහා යොමුවීමට පෙර, ස්වභාවික භාෂා සැකසීම යනු කුමක්ද සහ අප ඒ ගැන සැලකිලිමත් වන්නේ මන්දැයි ඉක්මන් දළ විශ්ලේෂණයක් කරමු.

## NLP යනු කුමක්ද?[[what-is-nlp]]

NLP යනු මානව භාෂාවට සම්බන්ධ සෑම දෙයක්ම අවබෝධ කර ගැනීම කෙරෙහි අවධානය යොමු කරන ලද වාග් විද්‍යාව සහ යන්ත්‍ර ඉගෙනීමේ ක්ෂේත්‍රයකි. NLP කාර්යයන්හි අරමුණ තනි වචන තනි තනිව තේරුම් ගැනීම පමණක් නොව, එම වචනවල සන්දර්භයද තේරුම් ගැනීමට හැකිවීමයි.

පහත දැක්වෙන්නේ එක් එක් උදාහරණ සමඟ පොදු NLP කාර්ය ලැයිස්තුවකි:

- **සම්පූර්ණ වාක්‍ය වර්ගීකරණය**: සමාලෝචනයක හැඟීම් ලබා ගැනීම, විද්‍යුත් තැපෑලක් අයාචිත තැපෑලක් දැයි හඳුනා ගැනීම, වාක්‍යයක් ව්‍යාකරණමය වශයෙන් නිවැරදිද යන්න හෝ වාක්‍ය දෙකක් තර්කානුකූලව සම්බන්ධද නැද්ද යන්න තීරණය කිරීම
- **වාක්‍යයක සෑම වචනයක්ම වර්ග කිරීම**: Identifying the grammatical components of a sentence (noun, verb, adjective), or the named entities (person, location, organization)
- **පාඨ අන්තර්ගතය උත්පාදනය කිරීම**: Completing a prompt with auto-generated text, filling in the blanks in a text with masked words
- **පාඨයකින් පිළිතුරක් උපුටා ගැනීම**: Given a question and a context, extracting the answer to the question based on the information provided in the context
- **ආදාන පාඨයකින් නව වාක්‍යයක් උත්පාදනය කිරීම**: පාඨයක් වෙනත් භාෂාවකට පරිවර්තනය කිරීම, පාඨයක් සාරාංශ කිරීම

NLP ලිඛිත පාඨවලට පමණක් සීමා නොවේ. එය ශ්‍රව්‍ය නියැදියක පිටපතක් හෝ රූපයක විස්තරයක් ජනනය කිරීම වැනි කථන හඳුනාගැනීමේ සහ පරිගණක දර්ශනයේ සංකීර්ණ අභියෝගවලට ද මුහුණ දෙයි.

## එය අභියෝගාත්මක වන්නේ ඇයි?[[why-is-it-challenging]]

පරිගණක මිනිසුන් ආකාරයට තොරතුරු සකසන්නේ නැත. උදාහරණයක් ලෙස, "මට බඩගිනියි" යන වාක්‍යය කියවන විට, එහි තේරුම අපට පහසුවෙන් තේරුම් ගත හැකිය. Similarly, given two sentences such as "I am hungry" and "I am sad," we're able to easily determine how similar they are. For machine learning (ML) models, such tasks are more difficult. The text needs to be processed in a way that enables the model to learn from it. And because language is complex, we need to think carefully about how this processing must be done. There has been a lot of research done on how to represent text, and we will look at some methods in the next chapter.
