# Ajuste fino con SFTTrainer

El ajuste fino supervisado (SFT) es un proceso que se usa principalmente para adaptar modelos de lenguaje preentrenados para que sigan instrucciones, participen en diálogos y usen formatos de salida específicos.

Esta página ofrece una guía paso a paso para ajustar el modelo [`deepseek-ai/DeepSeek-R1-Distill-Qwen-1.5B`](https://huggingface.co/deepseek-ai/DeepSeek-R1-Distill-Qwen-1.5B) usando [`SFTTrainer`](https://huggingface.co/docs/trl/en/sft_trainer).

## Cuándo usar SFT

Antes de entrar en la implementación, es importante entender cuándo SFT es la opción adecuada para tu proyecto.

> [!TIP]
> Considera SFT solo si:
> - Necesitas más rendimiento del que puede lograr el prompting
> - Tienes un caso de uso específico donde el costo de usar un modelo grande de propósito general supera el costo de ajustar un modelo más pequeño
> - Requieres formatos de salida especializados o conocimiento específico del dominio

### Control de la plantilla
SFT permite un control preciso sobre la estructura de salida del modelo.

### Adaptación al dominio
Al trabajar en dominios especializados, SFT ayuda a alinear el modelo con requisitos específicos del dominio.

## Configuración del entrenamiento

La configuración de `SFTTrainer` requiere tener en cuenta parámetros como:

1. `num_train_epochs`
2. `max_steps`
3. `per_device_train_batch_size`
4. `gradient_accumulation_steps`
5. `learning_rate`
6. `logging_steps`, `eval_steps` y `save_steps`
