Para empezar a usar Argilla, primero tendrás que configurar tu propia instancia. Después deberás instalar el SDK de Python para poder gestionar Argilla mediante código Python.

## Despliega la interfaz de Argilla

La forma más sencilla de configurar tu instancia de Argilla es mediante Hugging Face Spaces. Para crear tu Space de Argilla, solo tienes que seguir [este formulario](https://huggingface.co/new-space?template=argilla%2Fargilla-template-space). Si necesitas más orientación, consulta la [guía rápida de Argilla](https://docs.argilla.io/latest/getting_started/quickstart/).
<img src="https://huggingface.co/datasets/huggingface-course/documentation-images/resolve/main/en/chapter10/space_config.png" alt="Formulario de configuración del Space."/>

>[!WARNING]
> ⚠️ Puede que te convenga habilitar **Persistent storage** para que los datos no se pierdan si el Space se pausa o se reinicia.
> Puedes hacerlo desde Settings en tu Space.

Una vez que Argilla esté en marcha, podrás iniciar sesión con tus credenciales.

## Instala y conecta el SDK de Python

Ahora puedes ir a tu entorno de Python o notebook e instalar la librería `argilla`:

`!pip install argilla`

Conectémonos con nuestra instancia de Argilla. Para ello necesitarás la siguiente información:

- **La URL de tu API**: es la URL donde se está ejecutando Argilla. Si usas un Space, puedes abrirlo, hacer clic en los tres puntos de la esquina superior derecha, luego en "Embed this Space" y copiar la **Direct URL**. Debería verse como `https://<tu-usuario>.<nombre-del-space>.hf.space`.
- **Tu API key**: para obtenerla, inicia sesión en tu instancia de Argilla, ve a "My Settings" y copia la API key.
- **Tu token de HF**: si tu Space es privado, necesitarás un Access Token en tu cuenta de Hugging Face Hub con permisos de escritura.

```python
import argilla as rg

HF_TOKEN = "..."  # solo para Spaces privados

client = rg.Argilla(
    api_url="...",
    api_key="...",
    headers={"Authorization": f"Bearer {HF_TOKEN}"},  # solo para Spaces privados
)
```

Para comprobar que todo funciona correctamente, llamaremos a `me`. Esto debería devolver nuestro usuario:

```python
client.me
```

Si esto funcionó, tu instancia de Argilla está en marcha y ya estás conectado a ella. Bien.

Ahora podemos empezar cargando nuestro primer dataset en Argilla.
